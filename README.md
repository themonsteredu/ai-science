# 솔로몬 법정 · AI 과학수사 체험

더몬스터학원 · AI 과학수사 교실에서 쓰는 **단일 파일 웹 체험 앱**입니다.
설치 없이 브라우저에서 바로 실행되고, 이미지가 파일 안에 모두 들어 있어 **인터넷 없이도 동작**합니다.

## 사건

재판을 하루 앞둔 밤, 솔로몬 법정에서 핵심 증거인 **USB**가 사라졌다.
그날 밤 법정에 있었던 사람은 단 네 명 — 범인은 이 안에 있다.

## 체험 흐름

| 단계 | 내용 |
|---|---|
| 🎬 오프닝 | 사건 개요 시네마틱 브리핑 |
| 1. 현장 조사 | 오브젝트를 클릭해 단서 7개 수집 |
| 2. 과학 수사 | 지문 감식 — 채취 → 문양 분류 → 특징점 대조 |
| 3. AI 분석 | 증거 교차 분석 후 **직접 진범 지목** → 금고 암호 입력 |
| 4. 모의 재판 | 증거능력 심사 + 법정 상식 퀴즈 |
| 5. 최종 보고 | 수사 결과 및 배운 내용 정리 |

## 배우는 것

- 감이 아니라 **증거**로 판단하기 (현장에 있었다고 다 범인은 아니다)
- 지문 감식의 실제 절차 — 채취 → 문양 분류 → 특징점 대조
- 증거 하나로는 범인을 정할 수 없다 — **교차 분석으로 좁혀나가기**
- **증거재판주의** · **위법수집증거 배제** · **영장주의** · **무죄추정의 원칙**

## 실행 방법

- **온라인**: GitHub Pages 주소로 접속
- **오프라인**: `index.html` 파일을 받아 더블클릭 (USB·카톡으로 전달해도 그대로 동작)

## 기술

- 의존성 없는 단일 HTML 파일 (약 625KB)
- 외부 라이브러리·네트워크 요청 없음
- 이미지는 base64로 내장, 모바일·데스크톱 반응형
